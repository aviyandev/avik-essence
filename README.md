# avik/essence

Core utilities for the Avik framework.

## Purpose
Provides environment loading, configuration handling and minimal helper utilities.

## Includes
- Env loader
- Config repository
- Array and string helpers

## Design Rules
- No magic helpers
- No framework logic
- Minimal and stable API

## Stability
Frozen after v1.0.0 release.

## Requirements
- PHP 8.1+
- avik/seed ^1.0
